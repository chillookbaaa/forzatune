# 🏁 FH6 Tune Calculator

A lightweight single-file tuning calculator for **Forza Horizon 6**.  
Built to help create, compare, and revise car setups using car stats, upgraded build data, track profile, and real Forza test results.

## ✨ Main Features

- 🚗 **Car & Build Based Tuning**  
  Uses power, torque, weight, front weight %, drivetrain, tire compound, aero range, spring range, and gear count.

- 🔧 **Upgraded Stats / Build Snapshot**  
  Supports engine swaps and upgraded builds, so the tune follows the current build instead of stock car data only.

- 🗺️ **Track-Specific Formula**  
  Includes a custom **Hokubu Time Attack** profile based on baseline and X999 reference testing.

- 🧠 **Adaptive Tune Engine**  
  Calculates tires, gearing, alignment, anti-roll bars, springs, damping, aero, brakes, and differential based on car character and track needs.

- 📈 **Forza-Style Gear Graph**  
  Visualizes gear lines, usable top gear, target speed, and limiter risk.

- 📊 **Prediction & Compare System**  
  Compare predicted performance with real Forza Performance Monitor results like acceleration, braking, lateral G, top speed, aero balance, and aero efficiency.

- 🔁 **Revision System**  
  Apply multiple tuning revisions while keeping the latest tune state.

- 💾 **Save Slots**  
  Save build snapshots, tuning ranges, and Forza test results.

- 🌐 **Single HTML File**  
  Easy to upload to GitHub Pages and run directly in a browser.

## 🚀 How to Use

1. Open `index.html` in your browser or publish it with GitHub Pages.
2. Select a car, race type, and track.
3. Fill in upgraded build stats if the car has been modified.
4. Enter tuning ranges from the game.
5. Click **Hitung Tune** to generate a setup.
6. Test the car in Forza.
7. Enter Performance Monitor results.
8. Use **Compare** and **Revision** to improve the tune.

## 🎯 Project Goal

This project is not just a fixed preset generator.  
The goal is to create an adaptive tuning calculator that reacts to:

- 🏎️ car character
- ⚙️ engine and drivetrain
- 🛞 tire compound
- 🌦️ weather
- 🛣️ terrain and track type
- 📉 real test results
- 🔁 revision history
